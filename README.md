#  Hexacopter Drone - CDAC Workshop Project

##  Overview
This project was developed during a 1-week hands-on workshop at CDAC, where our team of 6 built a fully functional **hexacopter (6-motor UAV)** from scratch.

As an active team member, I contributed to the assembly, wiring, and calibration of the drone, and our team successfully achieved the **first flight among all groups**.

---

##  Objectives
- Understand multi-rotor drone architecture
- Gain hands-on experience in hardware assembly
- Learn flight controller configuration (Pixhawk)
- Perform real-time flight testing

---

##  Team Details
- Team of 6 members
- Collaborated on design, assembly, and testing
- Achieved **first successful flight in the workshop**

##  Hardware Components

| Component | Specification |
|----------|-------------|
| Motors | 1000 KV Brushless Motors ×6 |
| ESC | 30A Electronic Speed Controllers ×6 |
| Flight Controller | Pixhawk |
| Battery | 2200mAh Li-Po |
| Frame | Carbon Fiber Hexacopter Frame (6 arms + landing stand) |
| Power System | Power Distribution Board (PDB) |
| Sensors | GPS, IMU, Accelerometer, Gyroscope (inbuilt in Pixhawk) |
| Others | Buzzer |

---

##  Working Principle
The hexacopter uses **six brushless motors** to generate lift and control motion.

- Thrust is generated by rotating propellers  
- Orientation is controlled using IMU sensors  
- Stability is achieved via Pixhawk flight controller  
- GPS assists in positioning and navigation  

The 6-motor configuration provides:
- Better stability  
- Higher payload capability  
- Improved fault tolerance  

---

##  Build Process

### 1. Power System Setup
- Soldered ESCs to the Power Distribution Board (PDB)

### 2. Frame Assembly
- Attached landing stand  
- Mounted 6 motors on arms  
- Installed carbon fiber frame  

### 3. Wiring & Integration
- Connected ESCs to motors  
- Integrated all components with Pixhawk flight controller  
- Organized power and signal wiring  

### 4. Calibration
- Configured Pixhawk using supported software  
- Calibrated:
  - Accelerometer  
  - Gyroscope  
  - ESCs  

### 5. Final Setup
- Attached propellers  
- Connected battery  
- Performed pre-flight checks
