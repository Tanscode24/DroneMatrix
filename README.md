# DroneMatrix - CDAC Drone Workshop Project

##  Overview
This project was developed during a hands-on workshop at CDAC, where I designed and built a functional quadcopter drone from scratch.

The project focuses on understanding drone dynamics, hardware integration, and real-world flight testing.

---

##  Objectives
- Understand drone flight mechanics
- Assemble and integrate hardware components
- Perform real-time flight testing
- Learn basics of control systems

---

## Components Used
- Brushless Motors (1000KV)
- Electronic Speed Controllers (ESC)
- Flight Controller (KK2.1.5 / Pixhawk)
- Li-Po Battery
- Propellers
- Frame (F450 / custom)
- Transmitter & Receiver

---

##  Working Principle
The drone works on the principle of thrust generation using 4 motors. By varying motor speeds, lift and directional control are achieved.
